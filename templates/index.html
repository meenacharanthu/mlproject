Welcome to My Machine Learning Project

This project is designed to predict student performance based on various factors such as gender, race/ethnicity, parental level of education, lunch type, test preparation course, and scores in reading and writing.

Project Details

This project utilizes a machine learning pipeline to preprocess the input data, train multiple regression models, and select the best model based on performance metrics. The model is then used to make predictions on new data.

Key Features

  • Data Preprocessing: Handles missing values, encodes categorical variables, and scales numerical features.
  • Model Training: Trains multiple regression models including Random Forest, Decision Tree, Linear Regression, XGBoost, CatBoost, and AdaBoost.
  • Model Selection: Uses GridSearchCV to find the best hyperparameters and selects the best model based on R2 score.
  • Prediction: Provides predictions based on the trained model for new input data.

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-learn
  • XGBoost
  • CatBoost
  • Bootstrap
